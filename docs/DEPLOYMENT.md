# Deployment Guide

## Networks
- Base Mainnet: `forge script script/Deploy.s.sol --rpc-url base --broadcast --verify`
- Base Sepolia: `forge script script/Deploy.s.sol --rpc-url base_sepolia --broadcast --verify`

## Requirements
- Foundry installed
- `.env` with `PRIVATE_KEY` and `BASESCAN_API_KEY`
